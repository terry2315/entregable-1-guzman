Simulador de venta de autos

Para comenzar a utilizar el simulador de ventas de autos debes validar tus datos desde la consola del navegador llamando a la funcion validarUsuario(). Igresa los datos que seran solicitados mediante prompt. Debes cumplir con los requisitos. A continuacion se mostrara la lista de items en la consola del navegador. Al seleccionar tu item deberas hacer llamado de la function elegirItem("") e ingresar el id del mismo. Para finalizar podras llamar a la function verCarrito() para revisar tu lista de objetos.
